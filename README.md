# ICPADS26Benchmarks
This repository contains benchmark code, scripts, results, and related documents for a MASS C++ and MadMPI comparison prepared for an ICPADS 2026 submission

The project compares the performance, scalability, correctness, and programmability tradeoffs between MASS C++ and MadMPI using applied business and science workloads

For the full project background, methodology, workload descriptions, and organized results, see the report in this repository

Primary development was maintained in the original Bitbucket workspace (Equivalent copy to this repository: https://bitbucket.org/mass_application_developers/mass_cpp_appl/src/master/Benchmarks/ICPADS26/). This repository is used for consolidation and organization

ICPADS26 paper will be added when it is approved for submission
